The Lambda Calculus

First published Wed Dec 12, 2012

The \lambda-calculus is, at heart, a simple notation for functions and application.

To take a mathematical example, suppose we are given a simple polynomial such as x^2 -2\cdot x+5.

The central operation is function application:

\begin{align} (\lambda x[x^2 -2\cdot x+5])2 \rhd 2^2 -2\cdot 2+5 \\ = 4-4+5 \\ = 5 \end{align}

1. Introduction

The syntax of basic \lambda-calculus is quite sparse.

Definition The syntactic functions FV and BV (for free variable and bound variable) are defined recursively as follows:

FV(x) = {x} FV(λx.M) = FV(M) \ {x}

The \beta-reduction rule and \alpha-conversion are the core computation rules.

2. Syntax

The set of \lambda-terms is built from variables using abstraction \lambda x.M and application (MN).
